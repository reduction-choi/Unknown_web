# Unknown_web
굳이 온라인으로 할 필요 없잖아
그냥 내가 딜러보니까 내가 조작하면 되지 안그래?
조작은 전부 메인홀에서 해야하고 그냥 폰 들고 다니면서 상태 볼 수 있게 하는 용도정도로만 쓰자. (즉, 전부 관전자)

조작법:
어떤 칸이든 칸을 선택하면 24가지의 주사위 배치와 빈칸 총 25개 중 1개를 선택할 수 있다.
그리고 주사위 배치 선택 밑에 사람 이름 뭘로할지 넣어서 하면 될듯.
각 사진에 1~24까지 이름붙여서 그냥 표시하면 될 듯.

라운드는 그냥 옆에다가 표시할 수 있으면 하자.
